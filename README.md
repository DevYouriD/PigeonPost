# PigeonPost

Minecraft Paper server monitoring system with real-time player tracking and Discord notifications

---

### The project consists of two parts:

## Minecraft (paper)Server Plugin
A lightweight Paper plugin that listens to server events (player join/leave) and forwards them to external services.

## Node.js Backend Server
A Node.js service that receives these events, processes them, and exposes them via a simple API and Discord webhook notifications.

This allows real-time monitoring of player activity on a Minecraft server.
